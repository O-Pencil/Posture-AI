## 2. 实测数据

### 2.1 当前使用模型

| 项 | 值 |
| --- | --- |
| 模型 | **Qwen2.5-0.5B-Instruct-MNN**（taobao-mnn，INT4） |
| 体积 | ~550MB（含 `embeddings_bf16.bin`） |
| 存储路径 | `filesDir/mnn_models/qwen2.5-0.5b/` |
| 下载源 | `https://hf-mirror.com/taobao-mnn/Qwen2.5-0.5B-Instruct-MNN/resolve/main/` |
| 联调默认 | 模拟器/low-RAM 友好；SME2 验收时再切 Qwen3-1.7B |

**测试 Prompt（Settings MNN DEBUG）：**

```text
请用一句不超过30字的中文提醒我坐直，语气温和，不要医疗诊断。
```

### 2.2 arm64 模拟器（AVD）

| 项 | 结果 |
| --- | --- |
| 用途 | UI、下载、MNN 加载、指标面板 |
| `hw sme2` | false（虚拟 CPU） |
| `lib sme2` | true（APK 内 libMNN 已含 SME2 编译） |
| `backend` | CPU / NEON（非 SME2） |
| 推理输出 | 易出现乱码、`!!!!`（INT4 + 虚拟 CPU 限制，**非 bug**） |
| TPS | 可参考 BENCH，**不以模拟器 TPS 作为验收** |

> 换更小/更大模型**不能根治**模拟器乱码；更小模型或略减 OOM，更大模型更易 OOM。

### 2.3 真机（已验收，具体型号待补 adb 记录）

| 项 | 结果 |
| --- | --- |
| 模型 | Qwen2.5-0.5B（App 内下载） |
| `hw sme2` | **false** |
| `lib sme2` | **true** |
| `backend` | **CPU**（NEON 回退） |
| `decode TPS` | **~88.69**（INFER TEXT 一次采样） |
| `ttft` | ~0ms（warm-up 后/UI 四舍五入） |
| 样例输出 | 「等一下，请您坐在椅子上放松一会吧！（舒舒服服）」 |
| 结论 | **端侧 LLM 功能验收通过**；语义略偏「放松」但中文可读、无乱码 |

## 3. 模型下载体验（2026-06-16 新增）

- **断点续传**：状态文件 `mnn_models/.download_state.json`
- **后台下载**：下载逻辑在全局 `modelDownloadService`，切 Tab 不中断；顶部 `ModelDownloadBanner` 显示进度
- **退后台**：按 Home 后进程存活时继续；**系统杀进程**后下次打开 App **自动续传**
- **多模型**：Qwen2.5-0.5B / Qwen3-1.7B 可选；`.active` 标记当前推理模型
- **已安装检测**：完整/不完整/使用中；按钮「下载 / 继续 / 更换 / 设为此模型 / 删除」
